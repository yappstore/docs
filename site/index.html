    Create, build and distribute AI Applications with ease with The Yapp Store

    Getting Started About yappstore.ai © Yapp

    In a nutshell... ¶

    • Cross-Platform

      We build your applications for use across MacOS, Windows and Linux - ensuring your apps can be used by the most amount of people.

    • Monetization

      No Subscriptions, No payment-per-message, No taking a massive cut of your revenue. You sell your applications, and reap the rewards. We just take a humble fee for providing the service

    • Developer Focused

      We believe developers shouldn't be boxed into a single platform.
      We don't limit you to a specific UI format like a generic chat interface, you are free to use any web framework you like build your UI.

    • Native API

      AI applications can be very powerful when they have access to the right resources. The Yapp Store API allows your apps to interact with the users machine

    • Secure

      We handle the security aspects of distribution so you don't have to. All bundled applications are maintained in a sandboxed environment by default.

    • Free to use

      Publishing applications is free - allowing you, the developer, to minimize costs.
